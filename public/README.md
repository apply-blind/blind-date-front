# PWA 아이콘 안내

이 디렉토리에 다음 아이콘 파일들을 추가해주세요:

## 필수 아이콘
- `pwa-192x192.png` - 192x192 픽셀 앱 아이콘
- `pwa-512x512.png` - 512x512 픽셀 앱 아이콘
- `favicon.ico` - 파비콘
- `apple-touch-icon.png` - iOS용 아이콘 (180x180 권장)

## 아이콘 디자인 가이드
- **컨셉**: Toast(🍞) + Jam(🍓) 조합
- **메인 컬러**: #FFB4A2 (Warm Peach)
- **강조 컬러**: #FFD93D (Honey Gold), #FF6B9D (Jam Red)
- **배경**: #FFF8F0 (Cream)

## 아이콘 생성 도구
- https://www.pwabuilder.com/ (PWA 아이콘 생성기)
- https://realfavicongenerator.net/ (파비콘 생성기)
